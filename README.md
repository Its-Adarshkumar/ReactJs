# ReactJS Learning Project

This project is part of my journey to learn and explore advanced concepts in **ReactJS**.
It includes hands-on implementations of various features like hooks, state management, context API, component patterns, and more.
I’m passionate about web development and currently diving deeper into ReactJS.
Although I already have knowledge of ReactJS, I am currently learning some more advanced concepts to strengthen my skills.

## 🚀 Technologies Used

- ReactJS (with Create React App or Vite)
- JavaScript (ES6+)
- HTML/CSS
- React Router
- React Hooks (useState, useEffect, useContext, useRef, etc.)
- (Optional) Redux or Zustand for state management
- (Optional) TailwindCSS or styled-components for styling

## 📚 Topics Covered

- Functional Components
- React Hooks
- Conditional Rendering
- Lists and Keys
- Forms Handling
- Lifting State Up
- Context API
- Custom Hooks
- Routing with React Router
- State Management (Redux/Zustand)
- Performance Optimization (memo, useMemo, useCallback)
